Pass by value (값에 의한 전달)
메소드를 호출할 때 데이터를 복사해서 매개 변수에 넘기는 것

Pass by reference (참조에 의한 전달)
매개 변수가 메소드에 넘겨진 원본 변수를 직접 참조
메소드의 선언에서 ref 키워드를 매개 변수 앞에 붙여주고 메소드를 호출할 때 다시 ref 키워드를 매개 변수 앞에 붙여줌

ref return (참조 반환값)
메소드의 결과를 참조로 반환
ref 한정자를 이용해서 메소드를 선언하고, return 문이 반환하는 변수 앞에도 ref 키워드를 명시
반환하는 결과를 호출자가 참조로 넘겨받고 싶다면 결과를 담는 지역 변수와 호출할 메소드의 이름 앞에 ref 키워드를 위치

ref local (참조 지역 변수)
참조로 반환받은 결과를 담는 지역 변수
