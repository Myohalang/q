Loop Statement (반복문)
특정 조건을 만족하는 동안 코드 또는 코드 블록을 계속 반복해서 실행하도록 하는 문장

while
조건문이 참인 동안 코드를 반복 실행
조건식이 false 값을 가지지 못한다면 프로그램은 while 문에서 무한루프
while ( 조건식 )
	{
	반복 실행할 코드 블록;
	}

do while
조건식을 평가하기 전에 무조건 처음 한번은 코드를 실행
do
{
	반복 실행할 코드 블록;
}
while ( 조건식 );

for
조건문이 참인 동안 코드를 반복 실행
while 문보다 반복을 더 정교하게 제어
for ( 초기화식; 조건식; 반복식; )
{
	반복 실행할 코드 블럭;
}
초기화식
반복을 실행하기 전에 가장 먼저, 딱 한 번만 실행되는 코드
for 반복문에서 사용할 변수 등을 이곳에서 초기화
조건식
조건식의 결과가 false가 되면 반복을 중단
반복식
반복이 끝날 때마다 실행
반복식이 실행된 후에는 조건식이 실행되어 반복을 계속 진행할지를 판단
순서
초기화식 > 조건식 > 반복 실행할 코드 블럭 (True) > 반복식

foreach
배열(또는 컬렉션)을 순회하며 각 데이터 요소에 차례대로 접근
배열(도는 컬렉션)의 끝에 도달하면 자동으로 반복이 종료
foreach ( 데이터 형식 변수명 in 배열 또는 컬렉션 )
{
	반복 실행할 코드 블록;
}
foreach문이 한번 반복을 수행할 때마다 배열 또는 컬렉션의 요소를 차례대로 순회하면서 in 키워드 앞에 있는 변수에 담음
