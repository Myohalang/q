위의 오류를 해결하는 방법은 간단하다.

git remote rm origin을 입력해주고 다시 주소를 add해주면 된다.

git config --global user.name Myohalang  //이름

git config --global user.email Myohalang@naver.com "깃허브 메일주소"

mkdir ~/MyProject   // 로컬 디렉토리 만들고

cd ~/myproject      // 디렉토리로 들어가서

git init            // 깃 명령어를 사용할 수 있는 디렉토리로 만든다.

git status          // 현재 상태

git add 화일명.확장자  // 깃 주목 리스트에 화일을 추가

git add .           // 이 명령은 현재 디렉토리의 모든 화일을 추가할 수 있다.

git commit -m “현재형으로 설명” // 커밋해서 스냅샷을 찍는다.

git remote add origin https://github.com/Myohalang/q.git // 로컬과 원격 저장소를 연결

<-- 여기서 오류가 발생하면...

방금 위애서 언급한 git remote rm origin을 입력해준다.

그리고 다시 주소를 아래처럼 add 해줌

git remote add origin https://github.com/Myohalang/q.git // 로컬과 원격 저장소를 연결

git remote -v // 연결상태를 확인한다.

git push origin master // 깃허브로 푸시한다.
